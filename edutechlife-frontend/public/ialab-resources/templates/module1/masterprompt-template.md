## 🎯 Estructura del MasterPrompt

### 1. CONTEXTO Y ROL
```
Eres un experto en [ÁREA DE ESPECIALIZACIÓN] con [AÑOS] años de experiencia.
Tu objetivo es [OBJETIVO PRINCIPAL].
Tu audiencia es [TIPO DE AUDIENCIA] con [NIVEL DE CONOCIMIENTO].
```

### 2. OBJETIVOS ESPECÍFICOS
- [ ] Objetivo 1: [DESCRIPCIÓN]
- [ ] Objetivo 2: [DESCRIPCIÓN] 
- [ ] Objetivo 3: [DESCRIPCIÓN]

### 3. RESTRICCIONES Y PARÁMETROS
```
- Longitud máxima: [NÚMERO] palabras/párrafos
- Formato de salida: [FORMATO]
- Estilo de lenguaje: [ESTILO]
- Limitaciones técnicas: [LIMITACIONES]
- Consideraciones éticas: [CONSIDERACIONES]
```

### 4. ESTRUCTURA DE RESPUESTA
```
1. INTRODUCCIÓN: [QUÉ DEBE INCLUIR]
2. DESARROLLO: [ESTRUCTURA DE CONTENIDO]
3. CONCLUSIÓN: [ELEMENTOS FINALES]
4. RECOMENDACIONES: [ACCIONES SUGERIDAS]
```

### 5. TÉCNICAS AVANZADAS
- [ ] Chain-of-Thought: [APLICACIÓN]

### 6. EVALUACIÓN DE CALIDAD
```
- Criterio 1: [CRITERIO] - Peso: [%]

- Puntuación mínima aceptable: [PUNTAJE]/100
```

## 🛠️ Ejemplo Práctico

### Prompt para Análisis de Mercado:
```
Eres un analista de mercado senior con 10 años de experiencia en tecnología.
Analiza el mercado de [PRODUCTO/SERVICIO] considerando:
1. Tendencias actuales (2024-2025)
2. Principales competidores
3. Oportunidades de crecimiento
4. Riesgos potenciales

Formato: Reporte ejecutivo con datos concretos
Longitud: 500-700 palabras
Incluir: Estadísticas, gráficos mentales, recomendaciones accionables
```
