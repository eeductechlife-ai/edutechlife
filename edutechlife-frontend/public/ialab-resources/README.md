## 🎨 Paleta Corporativa

### Colores Principales
- **Petroleum:** `#2D7A94` - Para títulos y elementos principales
- **Corporate Blue:** `#4DA8C4` - Color corporativo principal
- **Mint:** `#66CCCC` - Para elementos secundarios y acentos
- **Soft Blue:** `#B2D8E5` - Para fondos y elementos sutiles
- **Dark Blue:** `#004B63` - Para contraste y énfasis

## 📊 Enfoque Pedagógico VAK

Todos los recursos siguen el enfoque VAK (Visual, Auditivo, Kinestésico) de EdutechLife:

### Visual (40%)
- Infografías con paleta corporativa
- Diagramas y flujos de proceso
- Presentaciones visualmente atractivas
- Resúmenes gráficos

### Auditivo (35%)
- Guiones para podcasts
- Explicaciones paso a paso
- Resúmenes en formato audio
- Instrucciones narrativas

### Kinestésico (25%)
- Plantillas descargables
- Ejercicios prácticos
- Checklists interactivas
- Guías de aplicación inmediata

## 🛠️ Uso de los Recursos

### Para Estudiantes
1. **Navegar por módulos:** Selecciona el módulo que estás estudiando
2. **Descargar plantillas:** Usa las plantillas para tus proyectos
3. **Seguir checklists:** Asegúrate de completar todos los ejercicios
4. **Aplicar conocimientos:** Usa los recursos en proyectos reales

### Para Instructores
1. **Material de clase:** Usa las infografías en presentaciones
2. **Ejercicios guiados:** Asigna las plantillas como tareas
3. **Evaluación:** Usa las checklists para evaluar progreso
4. **Personalización:** Adapta los recursos a necesidades específicas
